1. git status
            - Trạng thái của respo đang đứng
        2. git branch name_branch
            - tạo nhánh mới từ nhánh hiện tại
        3. git checkout name_branch
            - chuyển đến nhánh
        4. git add "file_name"
            - thêm file vào cam kết
        5. git commit -m "note"
            - tạo một cam kết khi chúng có một file hoặc nhiều file thay đổi
        6. git push origin name_branch
            - đẩy những thay đổi đã cam kết lên origin
